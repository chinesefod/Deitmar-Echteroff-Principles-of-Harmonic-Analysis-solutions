# Solutions to *Principles of Harmonic Analysis* (Deitmar–Echterhoff)

This repository contains worked solutions to exercises from

**Anton Deitmar and Siegfried Echterhoff — *Principles of Harmonic Analysis***
(Springer, Graduate Texts in Mathematics).

The goal of this project is to provide clear and complete solutions to selected exercises in harmonic analysis

* Locally compact abelian (LCA) groups
* Haar measure
* Pontryagin duality
* Fourier transforms on groups
* Characters and dual groups
* Convolution algebras
* Positive definite functions
* Representations of LCA groups
* Applications to number theory and analysis

The solutions are written with full mathematical detail and are intended for students studying graduate-level harmonic analysis.

## Format

Solutions are written in **LaTeX** and compiled to PDF.

Source files are included so readers can modify or extend the solutions.

## Purpose

This project was created while studying harmonic analysis independently and aims to serve as a reference for others working through the book.

## Disclaimer

These solutions are unofficial and may contain mistakes.
Corrections and improvements are welcome via pull requests.
