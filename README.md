# Solutions to *Principles of Harmonic Analysis* (Deitmar–Echterhoff)

This repository contains worked solutions to exercises from

**Anton Deitmar and Siegfried Echterhoff — *Principles of Harmonic Analysis***
(Springer, Graduate Texts in Mathematics).

These solutions are unofficial and may contain mistakes.

Suggested corrections and improvements are welcome via email: noellundstrm@gmail.com

## Format

Solutions are written in **LaTeX** and compiled to PDF.

Source files are included so readers can modify or extend the solutions.
